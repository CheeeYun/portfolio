# git 重新學習

- ### 抓遠端最新狀態
  _git fetch_
- ### 看版本有沒有差異
  _git status_
- ### 如果有落後拉下來

  _git pull_

- ### commit 類型前綴

  _commit -m ' feat: 新增功能 '_  
  _commit -m ' fix: 修復 bug '_  
  _commit -m ' style: 排版調整 '_  
  _commit -m ' refactor: 重構 '_

- ### 查看目前分支
  _git branch_
- ### 建立新分支(但不切過去)
  _git branch xxxx_
- ### 切換到新分支
  _git checkout xxx_
- ### 刪除本地分支
  _git branch -d xxx_
- ### 遠端追蹤分支
  _git branch -r_
- ### 清理遠端追蹤
  _git fetch --prune_  
  _git fetch -p_

# vue API

## watch & watchEffect

- `reative` 物件有深度追蹤
- `watch`不用 `deep:true` 就可以追蹤
- `watchEffect` 只要有用到的直就會追蹤
- 連續追蹤多個物件,可用陣列

```js
watch([count, skills], [newCount, newSkills], [oldCount, oldSkills]);
console.log(`count變了:${oldCount}->${newCount}`);
console.log(`skills變了:${oldSkills}->${newSkills}`);
```

## provide & inject

- 父->孫傳遞(用來跨層傳遞資料)
- 通常用 pinia 取代

## Teleport

- 把 DOM 元素移動到 body 下
- 用來解決 z-index 問題
- 用作彈窗、loading 之類
- CSS 搭配使用 : z-index 、 position:fixed

```js
<Teleport to="body">
    <div v-if="showModal" class="modal">
        <p>Modal內容{{ skillsSummary }}</p>
        <button @click="showModal=false">關閉</button>
    </div>
</Teleport>
```
